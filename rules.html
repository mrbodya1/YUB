Правила и механика челленджа

1. ОБЩИЕ ПОЛОЖЕНИЯ

Сроки проведения: 27 апреля — 17 мая (3 недели)

Участники соревнуются в беге, зарабатывают километры и внутреннюю валюту — ромашки. Можно влиять на результаты соперников с помощью способностей. Побеждает тот, кто наберёт больше всех очков в своём зачёте.

Раздельный зачёт:

  • Мужчины и женщины соревнуются отдельно
  • Рейтинг считается отдельно для каждого пола
  • Призовой фонд делится пропорционально количеству участников каждого пола

Время:

  • Всё считается по времени Екатеринбурга (YEKT, UTC+5)
  • Сутки начинаются в 00:00 YEKT
  • Все ограничения по времени работают по этому часовому поясу

2. РЕГИСТРАЦИЯ

Чтобы участвовать, нужно:

  1. Написать боту в личные сообщения команду /start или нажать «Начать»
  2. Бот автоматически определит имя, фамилию и пол из профиля ВК
  3. После регистрации участник получает начальный баланс 0 ромашек

Важно:

  • Один человек = один аккаунт ВК
  • Нельзя регистрироваться с фейковых страниц
  • Администратор может дисквалифицировать участника за нарушения

3. ТРЕНИРОВКИ

3.1. Требования к тренировке

Обязательные условия:

  • Минимальная дистанция: 3 км (целое число)
  • Без ограничения по времени
  • Фото: скриншот GPS-трека с видимой датой тренировки

Ограничения:

  • Только 1 тренировка в сутки
  • Сутки считаются с 00:00 до 23:59 по времени Екатеринбурга
  • Нельзя добавить тренировку задним числом

Что считается тренировкой: только бег на улице с GPS-треком (Strava, Nike Run Club, Garmin, Adidas Running и др.)

Что НЕ считается: бег на дорожке, спортивная ходьба, шаги из шагомера, велосипед, плавание, силовые тренировки.

3.2. Процесс добавления тренировки

  1. Нажать кнопку «Добавить тренировку» в личке с ботом
  2. Ввести дистанцию в километрах (целое число, минимум 3)
  3. Ввести время в минутах (целое число)
  4. Отправить фото скриншота GPS-трека с датой тренировки

Бот проверит: что сегодня ещё не было тренировки, дистанция ≥ 3 км, фото прикреплено.

3.3. Награда за тренировку

Базовая награда: 30 ромашек
Дневной бонус: +20 ромашек (до 22:00 YEKT)
Ночная тренировка: 30 ромашек (после 22:00 YEKT)

📌 Примеры

Тренировка в 10:00 → 30 + 20 = 50 🌼
Тренировка в 21:59 → 30 + 20 = 50 🌼
Тренировка в 22:00 → 30 🌼
Тренировка в 23:30 → 30 🌼

4. РОМАШКИ

Ромашки — внутренняя валюта челленджа. Используются для покупки способностей.

ИсточникКоличество
Тренировка (днём)50 ромашек
Тренировка (ночью)30 ромашек
Рулеткаслучайно, до 250 ромашек
Достиженияот 50 до 300 ромашек

На что тратятся: Удвоение, Щит, Разделение, Потоп, Обнуление.

⚠️ Важно

Ромашки нельзя передать другому участнику, обменять на реальные деньги. Баланс ромашек не влияет на итоговый рейтинг.

5. СПОСОБНОСТИ

Способности покупаются за ромашки в личном чате с ботом.

5.1. Удвоение

  • Цена: 150 🌼
  • Цель: своя последняя тренировка
  • Эффект: километраж ×2
  • Ограничения: нельзя дважды, только без других позитивных способностей

5.2. Щит

  • Цена: 100 🌼
  • Цель: своя последняя тренировка
  • Эффект: защита от негативных способностей
  • Ограничения: нельзя на уже атакованную/защищённую тренировку

5.3. Разделение

  • Цена: 150 🌼
  • Цель: последняя тренировка соперника
  • Эффект: километраж ÷2
  • Ограничения: цель должна быть без щита и без других атак

5.4. Потоп

  • Цена: 150 🌼
  • Цель: топ-5 лидеров твоего пола
  • Эффект: километраж −20% у каждой цели
  • Механика выбора: топ-5 рейтинга того же пола (исключая себя), с доступными тренировками без защиты

5.5. Обнуление

  • Цена: 200 🌼
  • Цель: последняя тренировка соперника
  • Эффект: километраж → 0
  • Ограничения: цель должна быть без щита и без других атак

5.6. Общие правила

  • На 1 тренировку — 1 позитивная (Удвоение) и 1 негативная способность
  • Щит считается негативной способностью (занимает слот, но не меняет км)
  • При попытке атаковать защищённую тренировку — ромашки НЕ списываются

8. РЕЙТИНГ И ОЧКИ

8.1. Текущий рейтинг

Обновляется в реальном времени, учитывает все модификаторы. Раздельный для мужчин и женщин.

8.2. Еженедельные очки

Когда: каждый понедельник в 00:01 YEKT за предыдущую неделю (пн-вс).

Как: сумма км → место → очки + бонусы.

12345678910
50454138363432302826

Бонусы (+10): макс. км, длиннейшая тренировка, лучший темп, больше всех способностей.

Очки суммируются за 3 недели.
